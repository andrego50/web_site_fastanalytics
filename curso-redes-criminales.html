IA/ML para Analítica de Redes Criminales

Curso introductorio con enfoque en Latinoamérica y el Caribe. No requiere conocimientos previos de estadística ni programación — solo disposición para aprender.

Quiero inscribirme

Objetivo

Familiarizarse y aplicar conceptos de la inteligencia artificial y el machine learning (aprendizaje automático) al análisis de factores de riesgo generados por redes criminales.

Dirigido a

Profesionales que deseen aprender sobre inteligencia artificial y machine learning. No se requiere conocimiento básico de estadística ni familiaridad con algún lenguaje de programación, pues es un curso introductorio — pero sí mucha disposición para aprender.

Fuerzas del orden

Policía, fiscalía, inteligencia militar y agencias de seguridad del Estado.

Seguridad y justicia

Seguridad privada, sistemas judiciales y organizaciones de justicia.

Banca y aseguradoras

Prevención de fraude, lavado de activos, compliance y gestión de riesgos.

Empresas y organizaciones

Cualquier organización que enfrente amenazas criminales, extorsión o fraude.

Al finalizar el curso podrás

1

Conocer el funcionamiento de una base de datos

Entender la estructura, tipos y lógica detrás de las bases de datos utilizadas en el análisis de redes criminales y factores de riesgo.

2

Extraer información de una base de datos

Consultar, filtrar y obtener los datos relevantes para alimentar modelos de análisis criminal.

3

Depurar un conjunto de datos

Identificar y corregir datos faltantes, duplicados, inconsistentes o erróneos para garantizar la calidad del análisis.

4

Describir gráfica y estadísticamente un conjunto de datos

Aplicar estadística descriptiva y crear visualizaciones que revelen patrones, tendencias y anomalías en los datos.

5

Construir un modelo de regresión o clasificación

Crear modelos predictivos de Machine Learning a partir de una base de datos para anticipar factores de riesgo criminal.

6

Visualizar y presentar información

Utilizar herramientas de visualización para presentar de forma clara y accionable la información obtenida del análisis de datos.
